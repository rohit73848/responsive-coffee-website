# ☕ Coffee Shop Website

A modern and fully responsive Coffee Shop Website built with HTML, CSS, and JavaScript. This project features a clean UI, mobile-friendly navigation, and an interactive testimonial slider powered by Swiper.js.

## ✨ Features

- Responsive design for all devices
- Sticky navigation bar
- Mobile menu with open/close functionality
- Smooth scrolling navigation
- Hero section
- About section
- Menu section
- Customer testimonials slider using Swiper.js
- Gallery section
- Contact section
- Footer with social links
- Modern and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Swiper.js
- Font Awesome

## 📱 Responsive Breakpoints

| Screen Width | Slides Per View |
|--------------|----------------|
| 0px+         | 1 Slide |
| 768px+       | 2 Slides |
| 1024px+      | 3 Slides |

## ⚙️ JavaScript Functionalities

### Mobile Navigation

- Opens and closes the sidebar menu.
- Automatically closes the menu when a navigation link is clicked.

### Testimonial Slider

Implemented using Swiper.js with:

- Infinite loop
- Navigation buttons
- Pagination bullets
- Dynamic bullets
- Grab cursor effect
- Responsive breakpoints
